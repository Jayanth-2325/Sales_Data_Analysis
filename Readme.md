# 📊 Sales Data Analysis Project

##  Description

This project focuses on analyzing supermarket sales data to identify trends, patterns, and business insights. The analysis is performed using Python, where the dataset is cleaned, processed, and visualized using different charts.

---

##  Tools & Technologies Used

* Python
* Pandas
* Matplotlib

---

##  Dataset

The dataset used is a **Supermarket Sales Dataset**, which includes:

* Date of purchase
* Product line (category)
* Total sales amount
* Quantity sold
* Gross income (profit)
* Customer details (Gender, City, Payment method)

---

## Analysis Performed

The following analyses are included in this project:

### 1. Monthly Sales Analysis

* File: `monthly_sales.py`
* Shows how sales vary across different months

### 2. Category-wise Sales Analysis

* File: `category_wise_sales.py`
* Compares sales across product categories

### 3. Profit Analysis

* File: `profit_analysis_sales.py`
* Analyzes profit generated by each category

### 4. Gender-wise Sales Analysis

* File: `gender_sales.py`
* Compares sales between male and female customers

### 5. Quantity Month-wise Analysis

* File: `quantity_month_sales.py`
* Shows demand trend across months

---

## Visualizations

This project includes:

* Bar charts for sales, profit, and quantity analysis
* Pie chart for gender-based sales distribution

---

##  How to Run the Project

1. Install required libraries:

   ```
   pip install pandas matplotlib
   ```

2. Place the dataset file:

   ```
   supermarket_sales.csv
   ```

   in the project folder

3. Run each file separately:

   ```
   python monthly_sales.py
   python category_wise_sales.py
   python profit_analysis_sales.py
   python gender_sales.py
   python quantity_month_sales.py
   ```
